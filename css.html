Qu'est-ce que le CSS ?

CSS est un langage de presentation de page web. Il est l'abreviation de "Cascading Style Sheets", ce qui signifie en francais "Feuilles de style en cascade". Il a ete introduit dans les annees 1996, pour ameliorer le rendu visuel des pages HTML. En effet, le langage CSS ne s'utilise qu'avec HTML ou XHTML. Son developpement est assure par le W3C (World Wide Web Consortium).

Comment l'utiliser ?

Il faut utiliser des navigateurs pour observer directement les resultats du code. Les plus communs sont Internet Explorer, Google Chrome, Safari et Firefox. Il est toutefois conseille de verifier son code sur plusieurs navigateurs, car ils possedent tous des differences de compatibilite, ce qui peut par exemple ne pas afficher les fonctionnalites recentes. En effet, Internet Explorer a longtemps ete en retard par rapport a ses conccurents dans le domaine du CSS, meme si depuis les dernieres mises a jour, ce retard tend a disparaitre. Mais rassurez-vous, tous les navigateurs ont leur defaut. Si un navigateur ne peut lire une instruction, il se contentera de l'ignorer. Il existe plusieurs manieres d'utiliser le langage CSS, je vais les presenter de celle que je juge la plus pertinente (en haut) a la moins pertinente (en bas):
  1. Comme le langage CSS est associe au langage HTML, il est envisageable d'utiliser un editeur de texte comme Geany ou Notepad++.
    Commencez par creer un fichier HTML (Comment creer un fichier HTML ?). Ouvrez votre editeur de texte, personellement, j'utilise Notepad++ ainsi que le navigateur Google Chrome. Reglez la langue (parametrage --> preference) et le langage (Langage --> C --> CSS) afin d'optimiser votre "comprehension du logiciel", c'est ce que j'appelle la "comprehension des options" et la coloration syntaxique automatique. Creez votre fichier (Fichier --> Nouveau ou CTRL+N), et enregistrez-le quelque part sous le format "css", c'est-a-dire "x.css". Par la suite, allez dans votre fichier HTML, trouvez la partie "< head >" du code, et notez : " < link rel="stylesheet" href ="x.css" />
    Vous venez de lier votre fichier CSS a votre fichier HTML! Sachez que l'on peut lier la meme page CSS pour plusieurs pages HTML, en procedant comme dit precedemment sur les pages HTML.
  2. La deuxieme methode pour utiliser du CSS dans son fichier consiste a inserer le code CSS dans une balise "< style > < / style >" qui se situe dans la balise "< head > < / head>".
  3. L'avant derniere methode consiste a mettre des attributs style directement dans les balises. Cela donne par exemple :
    " < p style="color=olive;"> Ce texte sera de couleur olive ! < / p > "
  4. Et enfin la derniere methode, la plus vieille et la moins pratique. Elle consiste a ne pas utiliser CSS du tout, et a decorer sa page en utilisant integralement le langage HTML. Peu pratique, le nombre de balise augmentait exponentiellement, et le programmeur finissait tot ou tard par se perdre. Cela donnait par exemple : "< p font color="#aab1c3" > Ce texte sera colore ! < / p >".

Quelques bases du langage CSS

Le langage CSS est compose de trois grands elements:
  1. Le nom de balise
    Il s'agit du nom de la balise HTML dont on cherche a changer l'apparence.
  2. Ce que l'on met a l'interieur
    Il s'agit des effets de style qui s'appliqueront a la balise. Il peut s'agit de couleurs, de mots colores ou, plus globalement, de fonds d'ecrans par exemple. Il faut entrer la propriete, ajouter ":" puis mettre la valeur que l'on veut afficher, et terminer par ";".
  3. Les accolades
    Situees sous le nom de la balise, et autour de(s) propriete(s) de CSS, elles lient le nom de la balise a ses attributs.
Un petit exemple :
Par exemple, pour colorier les titres h1 et h2 en rouge, on va utiliser la propriete "color". Cela s'ecrit :
h1,h2
{
color:red;
}

Maintenant que vous en savez plus a propos du fonctionnement du langage CSS, il est temps d'apprendre a s'en servir !
  • Inserer des commentaires
    Les commentaires n'apparaissent pas sur la page web, ils restent a l'interieur du code. Ils ont pour avantage d'aider le programmeur a se reperer, surtout lorsque le code est tres long et/ou tres complique. Cela peut egalement servir a aider un autre programmeur a se reperer. Il s'ecrivent sous la forme :
    "/* Ceci est un commentaire */"

  • Changer la taille du texte
    Il existe deux methodes pour changer la taille d'un texte.
    1. La taille absolue
      p
      {
      font-size: 16pix;
      }
      Cela permet de choisir la taille en pixel de ses caracteres.
    2. La taille relative
      p
      {
      font-size: small/medium/large/x.x em;
      }
      Ici, il ne faut choisir qu'une seule des propositions precedentes. "Small" correspond a une petite police, "medium" correspond a une police de taille normale, "large" correspond a une grande police, "x.x em" correspond a une taille qui varie en fonction des "x" (1 em correspond a une police de taille normale, quand ce nombre augmente, la taille augmente egalement).

  • Changer la police
    p
    {
    font-family: Arial,Verdana,etc... serif; /* L'ordinateur essaiera d'afficher Arial en premier, puis Verdana, et ainsi de suite... */
    }
    "serif" correspond a une police par defaut, qui est disponible quelque soit l'ordinateur.

  • Changer le style d'ecriture
    p
    {
    font-style: italic/oblique/normal/bold/underline/line-through/overline/blink ;
    }
    "italic" = italique, "oblique" = une sorte d'italique, "normal" = le texte est normal (ca peut servir a annuler une mise en italique par exemple), "bold" = gras, "underline" = souligne, "line-through" = barre, "overline" = sousligne mais au dessus, "blink" = clignotant.
    On peut en mettre plusieurs en meme temps, mais trop de details a la fois peut nuire a l'esthetique de la page web.

  • Changer le fond d'ecran
    body
    {
    background-color = aqua; /*Le fond d'ecran sera de couleur bleu clair */
    color = olive ; /*Le texte sera de couleur vert-olive */
    }
    Ici, le fond d'ecran est devenu noir et le texte est devenu vert-olive. Cela montre que meme si c'est la balise < body > qui est utilisee, le texte garde sa couleur verte-olive.

  • Appliquer une image de fond
    body
    {
    background-image: url("image.gif");
    background-attachment : fixed (fixe) / scroll (image qui defile) ; /*Deux options a propos du fond d'ecran */
    }
    Il faut placer le fichier image.gif dans le meme dossier que la page HTML et CSS.
  • Traiter une balise en particulier
    Je veux dire, comment faire pour mettre un paragraphe < p > en rouge quand tous ses paragraphes sont deja mis en bleu ? Pour cela, il est possible de rajouter a ce paragraphe un attribut "class". Cet attribut sert a attribuer un nom a ce paragraphe, que l'on pourra utiliser dans la feuille de style CSS. Dans le fichier CSS, on garde la meme syntaxe, et on ajoute un "." devant cet attribut. Cela donne par exemple :
    Dans le fichier HTML :
    < p > Ceci est un paragraphe qui restera en bleu < /p >
    < p class="ParagrapheRouge" > Ce petit bout de texte sera note en rouge. < /p >
    Dans le fichier CSS :
    p
    {
    color:blue;
    }

    .ParagrapheRouge
    {
    color:red;
    }

  • Traiter un mot en particulier
    Meme problematique que precedemment, peut-on peut changer un mot en vert quand le paragraphe qui l'entoure est ecrit en bleu ? La reponse est oui. Pour ce faire, on doit utiliser des balises dites "universelles" comme "< span > < / span>" ou "< div " < / div>".

    Dans le fichier HTML :
    < p > Ce mot va etre écrit en < span class="vert">vert< / span>, et seulement celui-là !< / p>

    Dans le fichier CSS :
    p
    {
    color:blue;
    }
    .vert
    {
    color:green;
    }

  • Selectionner une balise contenue dans une autre balise plus grande
    p strong
    {
    color : blue;
    }
    Cela selectionne toutes les balises < strong > situees dans les balises < p >. Retenez qu'on les ecrit du plus grand au plus petit ensemble d'ou le nom de CSS qui veut dire "Feuilles de style en cascade".
  • Selectionner une balise situee juste apres une autre
    h1 + p
    {
    color : olive;
    }
    Cela selectionne la premiere balise (ici un paragraphe) situee apres le titre h1. Encore une fois, on part du plus grand ensemble (h1).
