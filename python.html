Qu'est-ce que le Python ?

Python est un langage semi-interprete multi-paradigme(= maniere d'executer plusieurs procedures afin d'arriver a un resultat) cree par Guido Van Rossum vers les annees 1990. Il s'inspire du langage ABC pour creer Python, dont le nom "Python" vient de la serie "Monty Python".

Il est gratuit et fonctionne sur toutes les plateformes (Windows, Linux ou encore OSX). Aujourd'hui, les versions courantes sont les versions 2.7.3 et 3.2.3 (l'evolution
du langage est assuree par la Python Software Foundation.

Comment l'utiliser ?

Pour utiliser le langage Python, il faut un editeur de texte. Geany et Notepad++ conviennent, meme si d'autres editeurs specialises dans le langage Python comme Pyzo sont plus pratiques. Il est egalement possible d'aller sur le site officiel de Python : Le site officiel de Python. A partir de ce site, vous pourrez telecharger une application nommee "IDLE", qui permet de faire des scripts de Python.

Quelques commandes

Le contenu suivant est relativement peu detaille, il manque beaucoup de passage et les passages cites sont resumes tres brievement. Afin de limiter les erreurs, il est conseille de ne pas recopier mot a mot ce
qui est ecrit, et d'utiliser un editeur qui indente automatiquement (tant qu'on y est, il faut indenter en appuyant sur *TAB* ou plusieurs fois sur *ESPACE* a chaque fois que vous voyez ". .").
  • Pour definir une variable, il suffit de lui donner un nom, de mettre un "=" et de lui attribuer une valeur (3 pour int par exemple), (7.1 pour float par exemple) ou encore ("Hello" pour string par exemple).
  • On peut egalement faire des operations a l'aide de Python, comme ecrire dans la console "2 + 2". On peut egalement mettre des variables : "a - b" par exemple. Pour les differentes operations, il existe "+", "-", "*" pour multiplier, "/" pour diviser et bien d'autres.
  • Pour creer des structures conditionelles, il existe divers elements. Tout d'abord, il y a les instructions de controle (if,elif,else respectivement si,et si,sinon en anglais) qui permettent au
    programme de tourner tout seul. Par la suite, on trouve le bloc d'instructions, de la forme :
    "if x > 2:

  • Pour faire une boucle, il existe deux methodes.
    La premiere est la boucle "for k in range(x,y,z)"avec x une valeur comprise, y une valeur exclue et z le nombre de pas pour passer de x a y. Cela signifie en anglais "pour k de portee(x,y,z)".
    La seconde se traduit par "tant que", elle s'appelle "while". Apres avoir ajoute une instruction de controle, elle execute l'instruction. Attention, il faut mettre une instruction pour stopper la boucle
    sinon elle peut tourner indefinimment !

  • Il est possible de definir sa propre fonction. Pour cela, il faut ecrire "def NomDeLaFonction(x):". A la fin du bloc d'instruction, preferer le mot-clef "return" a la fonction "print".
    Un exemple de fonction:
    def max(a,b):
    . .if a>b:
    . . .return a
    . .return b
  • On peut bien entendu manipuler des fichiers. Pour creer un fichier, ecrire "f=open("Fichier.txt",'w'). Remplacer "Fichier.txt" par le chemin si ce fichier n'est pas dans le meme dossier que
    le programme Python. Il existe d'autres manipulations comme "f.read()" qui lit tout le fichier et bien sur "f.close()" qui permet de le fermer.
